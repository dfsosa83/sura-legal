Este reto técnico para el cargo de **Analista de Analítica Legal en la Gerencia de Asuntos Legales de SURA** busca evaluar tu capacidad para transformar datos en recomendaciones estratégicas, identificar riesgos y mejorar la eficiencia operativa mediante el análisis de información jurídica.

El ejercicio se divide en dos grandes retos:

**Resumen del Reto Técnico**

**Reto 1: Análisis de Datos Operativos****Parte 1 (Tutelas):** Debes analizar una base de datos sobre la gestión de tutelas para construir indicadores de oportunidad, cumplimiento de términos y riesgos operativos por regional o causal.**Parte 2 (Contratos):** Se solicita el análisis de una base de gestión contractual para identificar contratos críticos, riesgos de vencimiento y carga operativa por compañía.**Reto 2: Optimización de Procesos (Facturación)**Debes diagnosticar un proceso de facturación de abogados externos que actualmente es manual y descentralizado, para proponer controles y un nuevo flujo de trabajo automatizado y eficiente.

\--------------------------------------------------------------------------------

**Principales preguntas a resolver**

El reto especifica que tus análisis y herramientas (como tableros en Power BI o Excel) deben ser capaces de responder a las siguientes interrogantes:

**Sobre la gestión de Tutelas (Reto 1 - Parte 1):**

**¿Dónde se concentran los principales riesgos?****¿Qué etapas del proceso presentan mayores incumplimientos?****¿Qué variables impactan la oportunidad de respuesta?****¿Qué regionales, áreas o causales generan mayor presión operativa?****¿Qué casos deberían priorizarse?**

**Sobre la gestión Contractual (Reto 1 - Parte 2):**

**¿Cuál es el estado actual de la operación contractual?****¿Qué contratos requieren atención prioritaria y dónde se concentran los riesgos de vencimiento?****¿Qué compañías o tipos de contrato generan mayor carga operativa?****¿Existen concentraciones de contratos próximos a expirar?****¿Qué variables deberían monitorearse de forma permanente?**

**Sobre el Proceso de Facturación (Reto 2):**

**¿Qué controles concretos permitirían prevenir pérdidas de facturas, mejorar la eficiencia y asegurar la trazabilidad?****¿Cómo sería un flujo de trabajo estructurado desde la recepción hasta el pago?****¿Qué roles deben intervenir y qué tareas pueden automatizarse para reducir errores y duplicidad?**

**Entregables clave**

Para cada sección deberás entregar:

**Archivos de análisis** (Power BI, Excel, etc.) con visualizaciones e indicadores.**Documentos ejecutivos** (máximo 2 páginas) con hallazgos, riesgos y recomendaciones accionables.**Propuestas de mejora y automatización** para elevar la calidad del dato y la eficiencia operativa.
