# sura-legal

Este repositorio contiene una prueba técnica para un rol de analista del área legal en SURA Colombia.

## Contenido
- `data/dataset_1_litigios.csv`: dataset de litigios.
- `data/dataset_2_contratos.csv`: dataset de contratos y cumplimiento.
- `deliverables/legal_analyst_test.md`: desarrollo completo de la prueba, respuestas, mejoras e insights.
